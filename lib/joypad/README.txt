./lib/joypad

The joypad class is a wraper for RobotC's joystick implementation.

Features:
* Automaticly adjust joystick's output range from (-128,127) to (-100,100)
* Ability to scale the joystick's output, ex: 80%
* Adds dead zone for each analog stick
* Button constants for easier implementation, ex: BUTTON_X, BUTTON_DPAD_UP
